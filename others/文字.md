# ディレクトリ構造記述に使用する文字
## コピペ用
```
┣ ┠ ┝ ├
┫ ┨ ┥ ┤ 
│ ┃
─ ━
┌ ┏ ┓ ┐
└ ┗ ┛ ┘
```
## 変換用
MSIMEだと以下で変換できる
|読み|文字|
|:--|:--|
|たて|│ ┃|
|たてひだり|┫ ┨ ┥ ┤ |
|たてみぎ|┣ ┠ ┝ ├|
|ひだりうえ|┌ ┏ ┓ ┐|
|ひだりした|└ ┗ ┛ ┘|
|ふとわく|┗┻━┛ ┏┳━┓|
|ほそわく|└┴─┘ ┌┬─┐|
|まんなか|┼ ╋ ┿ ╂|
|みぎうえ|┐ ┓|
|みぎした|┘ ┛|
|よこ|─ ━|
|よこうえ|┻ ┷ ┸ ┴|
|よこした|┳　┯ ┰ ┬|
